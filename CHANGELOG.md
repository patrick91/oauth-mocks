CHANGELOG
=========

0.1.0 - 2026-07-10
------------------

OAuth Mocks is now available as a single installable package with a CLI that
runs the GitHub and Google OAuth mocks together.

Start both providers without cloning the repository:

```console
uvx oauth-mocks
```

This release was contributed by [@patrick91](https://github.com/patrick91) in [#1](https://github.com/patrick91/oauth-mocks/pull/1)
